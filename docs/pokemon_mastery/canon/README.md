# GSC Canon Lookup

Purpose: compact, source-linked GSC facts for precise retrieval during move
choice. These pages are small enough to load when the public board actually
asks for the topic; they are not a replacement for `live_core.md`.

Use them with the information tiers in `../heuristic_core/public_info_tiers.md`:
revealed facts can drive the line, strong priors can price branches, and
possible-only sets cannot anchor the main recommendation without a fallback.
