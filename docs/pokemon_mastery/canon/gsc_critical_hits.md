# GSC Canon: Critical Hits

Use when a line depends on crit risk, boosted damage, Curse mirrors, Reflect,
Light Screen, or one-hit survival.

## Route Rules

- GSC critical hits deal doubled damage.
- Critical hits do not mean "ignore all boosts" as a blanket rule. GSC crit
  damage uses a comparison of attacker and defender stages; the shortcut is:
  check the actual stage relationship before assuming a boost is ignored.
- Crit risk is a route tax. It should change recommendations when the
  non-crit line wins only by passing through repeated avoidable crit windows.
- Do not let low-probability crit fear override a line that is otherwise the
  only route to conversion. Name the fallback instead.
- For romhack advice, verify whether the local engine changed crit behavior
  before making a damage-threshold claim.

## Move-Choice Checks

1. Is survival or KO dependent on a crit, or does the line work without it?
2. Are boosts/screens relevant under GSC crit rules for this exact stage
   relationship?
3. Can a move shorten the crit window while preserving the route piece?
4. Is the crit branch a realistic top branch or only possible-only fear?
