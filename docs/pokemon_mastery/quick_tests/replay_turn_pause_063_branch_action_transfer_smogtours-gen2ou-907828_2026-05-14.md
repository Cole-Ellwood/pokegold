## Error Classes Found

- Branch-action still unstable: turns 2, 11, and 13 show that naming one branch
  is not enough; the move must cover the best branch, not just the visible
  active matchup.
- Over-cash: turn 7 repeated the urge to spend Cloyster when a hard answer
  preserved both board safety and the Explosion threat.
- Good transfer: turns 10 and 12 correctly acted on predicted switch branches
  with Raikou counter-switches.
- Support status timing: turn 9 showed Toxic can be the correct branch action
  when it changes a support piece's future job more than switching does.

## Reusable Lesson

Branch discipline needs a two-step check: name the branch, then ask whether the
chosen move actually wins that branch. A branch read that does not change the
move is just commentary.
