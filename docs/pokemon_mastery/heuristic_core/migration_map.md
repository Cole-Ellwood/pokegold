Before fresh answer freeze, use only `live_core.md`, the prompt, and the useful
tiny heuristic cards. After scoring, use this map to open the old card, review,
quick test, ledger row, or mechanics fixture that explains the miss.

reviews, quick tests, external research returns, and mechanics docs remain the
evidence archive. They are demoted from live decision context, not removed.
