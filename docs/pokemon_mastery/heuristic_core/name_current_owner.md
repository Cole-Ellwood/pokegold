# Heuristic: Name Current Owner

Status: live tiny card.

Use when: the visible matchup looks safe but the route owner is unclear.

Rule: name who owns the board before choosing a move.

Ask:
- What is our current converter or defender?
- What opponent piece stops that route?
- Does the active move remove, force, or improve against that piece?

Top move favors the owner that converts now, not the move that merely looks
comfortable into the active target.

Do not preserve or switch by habit if active pressure already changes the route.
