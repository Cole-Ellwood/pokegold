# Heuristic: Rest Curse Tempo Window

Status: live tiny card.

Use when: a boosted attacker, Curse user, phazer, or Explosion user faces a
Resting or likely-to-Rest bulky target.

Rule: solve the next two turns, not only the visible HP.

Ask:
- Who gains more from this Rest/sleep turn: our boost/phaze/trade route or
  their wake/Curse/attack route?
- Does attacking now change the next forced choice, or only feed a reset?
- Does boosting now change the next damage range or make their attack fail?
- If I guessed boost last turn, what changed that makes attack better now?
- If I guessed attack last turn, what changed that makes boost better now?
- Is the opponent on a Rest turn, sleeping action turn, wake/act turn, or able
  to Rest before our hit?
- If the helper marks `Rest sleep actions 2`, the Pokemon wakes and can act on
  the prompted turn; rank the awake move that converts before their reply.
- If Sleep Talk called Rest, did that start a fresh two-action Rest clock, and
  is this now the wake-and-act turn where status or KO must beat chip?
- If the RestTalk user can Sleep Talk, what do the attacking rolls do to the
  boosted passer or current owner before another attack converts?
- Does Baton Pass preserve a boost through those rolls better than one more
  damage click?

Top move must explain why boost, damage, phaze, or cash-out wins the two-turn
window.

Do not count chip as progress when it only lets the target Rest or wake into
the same stable board.
