## Turn 5 - Unused Sleep Source Before Active Pressure

Public state:
p1 Cloyster was at 51% with Spikes revealed, facing p2 Gengar at 100%. p2 had
not yet put anything to sleep. p1 had already slept p2 Zapdos, but that does
not block p2 from using Hypnosis.

Actual:
p1 switched Jynx. p2 Gengar used Hypnosis and slept Jynx.

Review:
My frozen top valued Surf into Gengar and treated sleep as a completed
subgame because p2 already had a sleeping Zapdos. That was a state mistake.
Sleep Clause is side-specific in this position: p1's sleep on Zapdos does not
remove p2's right to sleep one p1 Pokemon. Once p2 Gengar entered before using
sleep, p1 needed to name the sleep absorber before choosing active damage.

Reusable rule:
Before attacking a sleeper, ask which side has used sleep. If the opponent has
not slept anything yet and a sleep move is live, choose the absorber or explain
why the active piece can be slept without losing the route.

## Turn 7 - Substitute As Branch Action

Public state:
p1 Jolteon was at 92% against p2 Gengar at 100%. Gengar had revealed Hypnosis
and Thunderbolt. p2 had a sleeping Zapdos and a Snorlax, and the obvious
unrevealed answer class to Jolteon was Ground/Steel.

Actual:
p2 switched Steelix. p1 Jolteon used Substitute.

Review:
I named the receiver class but treated the action choice as attack or switch.
Substitute was the positive-selection move: it punished the receiver branch
without requiring the exact receiver, kept Jolteon active, and let p1 reveal
Hidden Power pressure afterward. This is not a hidden-info miss; it is a
branch-action miss. Once an immunity receiver is likely, setup or Substitute
can be the action that beats the branch.

Reusable rule:
When the expected branch is a defensive receiver, rank Substitute/setup beside
coverage and counter-handoff. Do not reduce the branch-action menu to only
"attack the current target" or "switch out."

## Turn 13 - Full-Health Gengar Is Not Low-Gengar Cash-Out

Public state:
p1 Zapdos was at 100% against p2 Gengar at 94%. Gengar had revealed Hypnosis
and Thunderbolt. p2 had a 60% Steelix, 82% Snorlax, sleeping RestTalk Zapdos,
and Spikes on its side.

Actual:
Gengar used Ice Punch, dropping Zapdos to 63%. Zapdos used Thunderbolt,
dropping Gengar to 43% and paralyzing it.

Review:
I over-applied the low-Gengar Explosion correction. Explosion was possible, but
this was not the same gate as a low, spent Gengar facing an irreplaceable
Zapdos. Full-health Gengar still had active coverage that pressured Zapdos,
was not forced to cash out, and could still keep value after attacking. The
correct first question was whether active coverage plus later paralysis or
switch pressure improved the route more than a one-time trade.

Reusable rule:
Do not promote Explosion from possible to primary just because the user is
Gengar. It needs the normal cash-out checks: prior job delivered, exact target,
delay loses the target, no cheap absorber, and active coverage does not already
improve the route.
