# Worked Examples

Purpose: short applications of the cookbook and live-turn template to concrete
positions. These are study artifacts, not a sealed benchmark set.

Coverage status:

- Every boss route map in `../boss_route_maps/` currently has a matching
  `*_pre_battle_route_sheet.md` worked example here.

Use them to practice:

- applying public state without relying on species slogans;
- naming the route and worst plausible branch;
- checking whether the final recommendation matches hidden oracle data after
  the answer is written;
- extracting one reusable recipe without bloating the notebook.
