# Worked Example: Sequential Converter Pressure

Source review: `../reviews/2026-05-13_smogtours-gen4ou-670049.md`

Source replay: https://replay.pokemonshowdown.com/smogtours-gen4ou-670049

Source context:
https://www.smogon.com/forums/threads/dpp-ou-rate-my-team.3716857/

Format: DPP OU. This is a strategic transfer example, not GSC or romhack
mechanics proof.

## Position Pattern

```text
Opponent side:
The first converter has forced a control response.
The control response stopped the immediate route.
However, the answer took damage, status, hazard tax, PP loss, or bad position.
The opponent still has a second converter or trade piece.

Our side:
We can say how we stopped the first route.
We must decide whether the same answer also covers the next route.
```

## Candidate Move Classes

Best when available:

- Deny the next converter while preserving the answer already spent on the
  first converter.
- Convert the temporary control into a KO, forced switch, safe entry, phaze, or
  sacrifice before the opponent handoff arrives.
- Use a trade only if it removes the piece that would inherit the pressured
  board.

Acceptable:

- Cash out a low-HP answer if it has completed its job and creates a clean
  entry for the next player route.
- Switch to the backup answer if the first answer no longer survives the second
  converter's entry.

Wrong:

- Treating the first stopped setup route as the end of the fight.
- Healing or scouting while the opponent's second converter enters for free.
- Sacrificing a piece because it is low without naming which later route the
  trade opens or closes.

## Rule

After any successful anti-setup or anti-converter move, immediately fill this
out:

```text
First converter stopped:
Answer used:
Cost paid by that answer:
Opponent piece that inherits next:
Our remaining answer to that piece:
Move that prevents a free handoff:
Abandon condition:
```

The correct next move is rarely "continue the old plan." It is the move that
prevents the opponent from turning the stopped route into a support ladder for
the next route.

## Boss-Battle Transfer

Use this against staged boss teams:

1. Name the visible converter.
2. Name the support or damage it leaves behind even if stopped.
3. Name the next boss piece that benefits from that residue.
4. Check whether the current player answer is still usable for that next piece.
5. If not, hand off before the boss handoff becomes forced.

This is especially important against Gym Leader Lab bosses whose rosters create
several route families instead of one ace. Winning the current exchange is not
enough if the answer that wins it is the same answer needed for the next route.
