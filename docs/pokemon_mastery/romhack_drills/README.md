# Romhack Drill Packets

Purpose: machine-readable and compact prompt sets for transferring Pokemon/GSC
mastery into this romhack's mechanics and boss roster context.

These are practice and regression drills unless a packet explicitly says it is
validation. Source-derived drills can prove coverage of a concept, but they do
not prove live play mastery until answered, scored, and checked against
fresh/semi-blind boss states or runtime fixtures.

Current packet:

- `transfer_drills_001.jsonl`: first romhack transfer packet covering Air
  Balloon, three-layer Spikes, Choice items, Life Orb, Eviolite / EVOLITE,
  Rocky Helmet/contact, Dragon Dance/Outrage, RestTalk, Explosion/cash-out, and
  local type rewrites.
- `transfer_drills_001_scores.jsonl`: scored action-class answers for the first
  packet.
- `transfer_drills_001_results.md`: packet-level metrics and next bottleneck.
