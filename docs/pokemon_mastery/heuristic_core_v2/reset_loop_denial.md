# Heuristic: Reset Loop Denial

Status: live tiny card.

Use when: Spikes, Rapid Spin, Rest, Recover, Heal Bell, phazing, Sleep Talk,
Substitute, or Baton Pass may reset the route.

Rule: progress counts only if the opponent cannot reset it for free.

Ask:
- Who resets: spinner, setter, Recover user, RestTalker, phazer, passer?
- What denies the loop: pressure, Toxic, Pursuit, phaze, cash-out, handoff?
- Does the current move convert before the reset happens?

Top move denies or exploits the reset loop, not the move that repeats unstable
progress.

Do not treat cleared hazards, chip, or a sleeping target as stable by default.
