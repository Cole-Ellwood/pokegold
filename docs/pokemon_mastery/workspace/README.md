# Pokemon Mastery Workspace

Purpose: preserve measurement provenance and raw research without making it
default battle-time context.

This directory is intentionally outside the live playbook path. Files here are
useful for audits, score trends, postmortems, and source reconstruction, but
they should not be loaded before a fresh unseen move answer is frozen.

## Contents

| Path | Role |
| --- | --- |
| `quick_tests/` | Scored transfer packets, probes, and post-oracle summaries. |
| `pro_notes/` | External method notes and process guidance. |
| `external_research_returns/` | Raw research returns that require filtering before adoption. |
| `measurement_reports/` | Progress-report artifacts. |
| `battle_captures/` | Real boss-attempt evidence and capture notes. |

Preserve these files. Promote only repeated, measured lessons back into
`../heuristic_core/`, `../policy_cards/`, or compact `../canon/` pages.
