## Reusable Lesson

Exact removal is not just damage range. It includes priority, speed, status,
and whether the self-KO user moves first. A paralyzed Electric that would KO
with Thunder still has not removed Shellder before Explosion if Shellder acts
first.

Against Toxic plus Recover, ask whether recovery still resets progress. Once
the toxic counter outpaces Recover, setup or preservation can be the route
converter because the poisoned target must spend turns healing and still dies.
Attack immediately only if it changes the next forced choice better than the
timer-backed setup.
