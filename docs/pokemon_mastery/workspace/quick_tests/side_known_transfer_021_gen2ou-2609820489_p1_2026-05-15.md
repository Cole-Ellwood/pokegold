## Reusable Lesson

Growth/Baton Pass turns need a three-way timing check:

1. Boost-now if both current Pokemon survive and the receiver route improves.
2. Attack-now if boosted coverage changes the active target before it can pass,
   explode, status, or reset.
3. Pass-now only when the named receiver survives the incoming hit and owns the
   next board better than the passer.
